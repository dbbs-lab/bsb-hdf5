# bsb-hdf5

`bsb-hdf5` is a plugin of the [BSB](https://github.com/dbbs-lab/bsb) (see also 
[bsb-core](https://github.com/dbbs-lab/bsb-core)). 
It contains the storage engine allowing the user to save and load their models' 
in HDF5 files.
